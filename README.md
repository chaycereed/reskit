# reskit

**reskit** is a lightweight command-line toolkit that helps you start new research projects with clean structure, clear documentation, and reproducible defaults. It automatically generates organized folders, a standardized README, and a ready-to-use analysis notebook template.

---

## Features

- Generates a complete, best-practice research project structure  
- Auto-creates a standardized project README  
- Includes a starter analysis notebook with seeds set  
- Encourages reproducible workflows from the beginning  
- Simple CLI interface: `reskit init <project-name>`
